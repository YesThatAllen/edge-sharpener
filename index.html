Edge-sharpener

A tool to optimize how LightSpeed and AccountEdge work
 

Export types

There are five types of files exported by LightSpeed Pro:

Inventory Adjustments

Memo re-organization for clarity & easier sorting

Miscellaneous Purchases

These transactions (which match LightSpeed Pro's Supplier Invoices) are simply transferred without modification.

Payments

Rather than have payments show up as journal entries, which cannot be included on deposit slips, the Edge Sharpener solution converts all exported payments from LightSpeed Pro into proper received payments in AccountEdge. The Memo includes the Customer Name, as well as the document number on which the payment was taken.

This allows for clearer delineation of payment types, and improved reporting.

There is no current mechanism for ensuring Customers in LightSpeed Pro are synced with AccountEdge, so Edge Sharpener sets Payor for all payments is a single "Imported Payment" Payor.

Sales

Memo re-organization for clarity & easier sorting.

Transfers

Memo re-organization for clarity & easier sorting.

Requirements:

Export Folder

Defining a certain folder where LightSpeed will save all 5 export files for a given session.

Transfer Folder

A certain folder where the Edge Sharpener will save converted files. AccountEdge will import from this folder.

LaunchAgent

A plist which watches the Export folder

Edge Sharpener Script

The heart of this solution

Optional, recommended

Dropbox account

It is frequently desired to have the imports happen on computer which is separate from the Export computer. In this case, having the Transfer Folder in a Dropbox account can be beneficial
